YouTube Video Combined Uploader Project

This project was to use the YouTube API to create a video uploading tool that would allow you to combine multiple videos into one. The original motive behind the development was to create videos that featured their own URL using this technique, however it has been expanded for more general use.
With this, you can combine two videos binary-wise mid-upload and create interesting effects, most notably the original reason this was made was to allow for your video to contain its own url.
No private data is collected by this application, certain private data is accessed by the application to facilitate the uploading of videos, however any data accessed is not stored remotely and is only accessible to you. The privacy disclaimer as required by YouTube is available here.
Due to the design of the code, the full code may only be released to private, trusted, individuals. However, a public version of the code is available on Github or in a zip file here.
